# El Radio app releases

Publiczny kanal dystrybucji aplikacji El Radio.

To repozytorium zawiera tylko gotowe pliki wydan, manifest aktualizacji i publiczne dane potrzebne aplikacji w czasie dzialania. Nie zawiera kodu zrodlowego aplikacji.

## Rolling release

- Tag: `latest-build`
- Android: `EL-Radio.apk`
- iPhone: `EL-Radio-unsigned.ipa`
- Manifest: `EL-Radio-release.json`

## Publiczne dane

- `data/facebook-feed.json` - cache publicznych aktualnosci z profilu El Radio.
- `assets/elradio-logo.png` - logo uzywane przez aplikacje i powiadomienia.
- `assets/elradio-logo-top.png` - logo naglowkowe aplikacji.

Wszystkie prawa zastrzezone. Pliki sa udostepnione wylacznie do instalacji i dzialania aplikacji El Radio.
